# Titanic-Dataset-Data-Analysis
This repository contains a comprehensive data analysis of the Titanic dataset using Python. The project focuses on exploring passenger data to identify patterns and factors that influenced survival during the Titanic disaster.
Using tools such as Pandas, NumPy, Matplotlib, and Seaborn, the dataset is cleaned, processed, and analyzed to generate meaningful insights. Key features such as age, gender, passenger class, family size, and fare are examined to understand their impact on survival rates.

The project includes exploratory data analysis (EDA), data visualization, feature engineering, and statistical insights to highlight important trends within the dataset. The goal of this repository is to demonstrate practical data analytics skills, data preprocessing techniques, and visualization methods commonly used in real-world data science projects.
